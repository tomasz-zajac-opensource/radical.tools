# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Architecture Hub — browse and import 70+ curated architecture concepts
- AI assistant integration (OpenAI, Anthropic, Gemini, Ollama)
- Smart Layout pipeline with SA-based auto-layout
- Metamodel editor — customise node types, relation types, and constraints
- Milestone-based time travel (named undo/redo snapshots)
- Multiple view types: Canvas, Matrix, Sequence, Treemap, Table, Wiki
- Presentation mode with fullscreen slides
- C4 model elements: Person, Software System, Container, Component, Database, Web App, Queue
- Domain & Governance elements: Domain, ADR, Fitness Function, Requirement, Blueprint
- Multiple layout engines: ELK (hierarchical/layered/force), webcola, Smart Layout
- PNG/SVG export, JSON save/load
- Electron desktop app + browser mode
